CSS란?

CSS(Cascading Style Sheets)란 "Style sheet" 언어로 웹페이지를 꾸미려고 작성하는 코드이다. HTML 문서에 있는 요소들에 선택적으로 스타일을 적용할 수 있다.

CSS 주석표시

css 파일의 내용이 많아지면 주석을 활용하여 관리하는 것이 좋다.

주석 표시하는 법 (단축키 : Ctrl + /) 
/* CSS 주석 */

CSS 선언 방법

내부 스타일

HTML파일 안에 <style>태그로 작성하는 방식이다. HTML, CSS, JS 파일을 나누어 관리하는 것이 일반적이므로 자주 사용되지 않는 방식이다.

외부 스타일

link 태그를 이용하여 외부 CSS문서를 가져외서 연결하는 방식이다. 실무에서 주로 사용하는 방식으로 대부분의 프로젝트에서 이 방식을 사용한다.

<link rel="stylesheet" href="../css/main.css">

인라인 스타일

요소의 style 속성에 직접 스타일을 작성하는 방식이다. 우선순위상 우선하기 때문에 공통기능에 대해서 따로 파일로 관리하는 경우 변경이 불가능 하기 때문에 유지 보수에 어려움이 있다.

<div style="css">
        ...
</div>

id/class 차이점

ID CLASS
차이점 한 페이지에 하나의 정의로 하나의 태그만 사용할 수 있다. 한페이지에 반복적으로 사용되는 스타일을 정의할 수 있다.
스타일 적용법 #ID명으로 적용시킴 .CLASS명으로 적용시킴(둘 이상 사용시 class= "class1 class2" 와 같이 띄어쓰기로 구분)
주의할 점 ID 또는 CLASS를 여러 개 가질 때 스타일이 중복되지 않도록 해야 한다.

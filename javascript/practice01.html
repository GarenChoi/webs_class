1.1부터 100까지 출력하기

1.1 while 문을 이용해서 1부터 100까지 출력하는 함수를 작성해주세요.

{
    let num = 1

    while(num<=100){
        console.log(num)
        num++;
    }
}

1.2 for 문을 이용해서 1부터 100까지 출력하는 함수를 작성해주세요.

{
    for(let i = 1; i<=100; i++){
        console.log(i)
    }
}
